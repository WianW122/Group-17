# Play It Forward – Supply Chain App

A web platform connecting donor stores and nonprofits to manage sports equipment donations.

## Features
- Log donations
- Schedule pickups
- Track deliveries
- Submit feedback

## Setup
1. Clone the repo
2. Run `npm install`
3. Start the server with `npm start`
